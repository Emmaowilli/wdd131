Coach Emmanuel Inc.

Site Name

Name: Coach Emmanuel Inc.

Reason: The name establishes a professional coaching brand led by Emmanuel, emphasizing personal and career growth in Kampala, Uganda. It’s concise, trustworthy, and appeals to clients seeking transformative coaching.

Site Purpose

The website promotes Coach Emmanuel Inc.’s coaching services, offering details on one-on-one and group sessions, client testimonials, and a contact form for booking consultations. It aims to engage visitors, provide resources for goal-setting, and build trust through success stories.

Target Audience Scenarios

  • What coaching programs are available to improve my leadership skills for a corporate role?
  • How can I contact Coach Emmanuel to start a personal development plan?

Color Schema

  • Navy Blue (#1B263B): Used for headers, navigation, and backgrounds to convey professionalism and trust.
  • Soft Gold (#E6B800): Used for buttons, links, and accents to inspire motivation and warmth.

Typography

  • Roboto:
    • Bold (700): Headings (h1, h2, h3) for emphasis.
    • Regular (400): Body text, paragraphs, and lists for readability.

Wireframes

Mobile View

The mobile layout is a single-column design optimized for small screens (320–600px):

  • Header: Logo centered, site title below.
  • Navigation: Hamburger menu (collapsible).
  • Main: Greeting text, mission statement, call-to-action button.
  • Footer: Contact link and copyright.

Desktop View

The desktop layout uses a wider design for larger screens (1024px+):

  • Header: Logo left, navigation links right.
  • Main: Hero section with image on right, text and button on left.
  • Footer: Full-width with contact and social links.
